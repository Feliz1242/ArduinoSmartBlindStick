# ArduinoSmartBlindStick
Arduino Project – Smart Blind Stick with Ultrasonic Sensor

This project is designed to make a smart blind stick with an ultrasonic sensor and Arduino. For more information, visit https://eddragon.com/smart-blind-stick-with-ultrasonic-sensor/.
